# 🧩 fakeapi

---

## 🚀 설치 (Installation)

```bash
npm install
```

---

## 🧑‍💻 개발 모드 실행 (Development)

```bash
npm run tauri dev
```

---

## 🏗️ 빌드 (Build)

```bash
npm run tauri build
```

> 결과물은 `src-tauri/target/` 디렉터리에 생성
